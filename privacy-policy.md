# Privacy Policy

_Last updated: November 25, 2025 _

## Overview

This browser extension is designed to provide a fast and keyboard-driven interface for switching and searching between open tabs. It does **not** collect, store, or transmit any personal or sensitive user data.

We respect your privacy and follow all Chrome Web Store Developer Program Policies.

---

## Information We Collect

The extension does **not** collect, store, or send any personally identifiable information.

Specifically:

- We do **not** track browsing history.
- We do **not** log website content.
- We do **not** store personal data on remote servers.
- We do **not** transmit data to third parties.

---

## Permissions Usage

This extension uses a small set of Chrome permissions purely for functional purposes.

### `tabs`

Used to list and display information about open tabs (title, URL, favicon) so the user can switch between them. No tab content is read.

### `activeTab`

Used to detect and highlight the currently active tab.

### `background`

Used to maintain keyboard shortcuts and enable smooth switching even when no popup is open.

### `storage`

Used only to store local UI preferences such as theme and shortcut configuration.

### `scripting`

Used for minimal scripting tasks required to manage the tab switcher UI. It does not read or modify webpage content.

### No Remote Code

The extension does not load or execute remote code.
